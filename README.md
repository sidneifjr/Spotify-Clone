## Spotify Clone

A attempt at making an SPA, using:

- Styled Components (for general styling);

- Stitches (for styling Radix UI components);

- Styled Icons;

- Radix UI;

- React/Hooks/Context API;

- Router;

- Typescript;

- Vite.

Deployed with Vercel.

### Development

First, run:

``yarn``

Then:

``yarn run dev``
